# Bus Management System

## About

This app helps bus passengers view timetable, select route they want and see live bus location.

## Features

-User login and registration
-Bus company selection
-View timetable
-Route selection
-live bus tracking
-Crowdsourced location sharing

## Technologies Used

Frontend:
-HTML
-CSS
-JavaScript

Backend
-FastAPI(Python)
